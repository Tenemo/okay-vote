# okay-vote web

Browser application for [okay.vote](https://okay.vote).

## local development

```bash
pnpm --filter @okay-vote/web start
```

## build

```bash
pnpm --filter @okay-vote/web build
```

For Netlify deploys, set `API_BASE_URL` to the public Railway API origin.
